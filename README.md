# 📊 Customer Retention & Churn Analysis

## 📌 Project Overview

This project focuses on analyzing customer churn and retention behavior for a subscription-based business using the Telco Customer Churn dataset.

The objective of this project is to identify:
- customer churn patterns
- key retention drivers
- customer lifetime trends
- high-risk customer segments
- actionable business recommendations

This project was completed as part of the **Data Science & Analytics Internship – Task 2 (2026)** by Future Interns.

---

# 🎯 Business Problem

Customer churn directly impacts business revenue and growth.  

The goal of this analysis is to help businesses understand:
- Why customers leave the platform
- Which customer groups are most likely to churn
- How customer behavior affects retention
- What strategies can improve customer loyalty and reduce churn

This analysis reflects real-world retention analytics used in SaaS and subscription-based companies.

# 📁 Dataset

### Dataset Used:
Telco Customer Churn Dataset

### Source:
https://www.kaggle.com/datasets/blastchar/telco-customer-churn

The dataset contains customer demographic details, subscription services, billing information, tenure, and churn status.

---

# 🧹 Data Cleaning & Preparation

The dataset was cleaned and prepared using Python and Pandas.

### Data preprocessing steps:
- Removed missing and blank values
- Corrected data types
- Converted TotalCharges to numeric format
- Handled null values
- Standardized column names
- Created Customer Lifetime Value (CLV)
- Created spend category segmentation
- Prepared data for Power BI dashboard analysis

---

# 📊 Dashboard Features

The interactive Power BI dashboard includes:

- Churn Rate KPI
- Active Customers KPI
- Customer Lifetime Value (CLV)
- Contract-based churn analysis
- Payment method analysis
- Internet service churn trends
- Customer tenure analysis
- Senior citizen churn analysis
- Interactive slicers and filters
- Business insights and recommendations

---

# 📈 Key Business Insights

- Overall churn rate is **26.5%**
- Month-to-month customers show the highest churn
- Electronic check users have higher churn rates
- Fiber optic users show relatively high churn behavior
- Customers with longer tenure are more loyal
- Active customers have higher CLV than churned customers
- Senior citizens show higher churn rates
- Gender has minimal impact on churn behavior

---

# 📌 Final Conclusion

Customer churn is mainly influenced by:
- contract type
- payment method
- internet service
- tenure
- customer lifetime value

The analysis suggests that businesses can reduce churn by:
- promoting long-term contracts
- improving customer experience
- encouraging automatic payment methods
- focusing on early customer engagement
- retaining high-value customers

---

# 💡 Business Recommendations

### Recommended strategies to reduce churn:
- Offer discounts for long-term contracts
- Improve onboarding experience for new customers
- Encourage automatic payment methods
- Provide personalized retention offers
- Improve support for high-risk customers
- Focus on retaining high-value customers
